# webpack-static-multiple-pages

This is a webpack template for build multiple static pages website

### feature

* Multiple Pages
* SEO
* Easy To Use
* Webpack5


### install

```shell 
npm i
```
or
```shell
npm install
```

### development

```shell
npm run serve
```

### building

```shell
npm run build
```

### preview

```shell
npm run preview
```
